# WebXR AR-сцена: дві моделі ліворуч і праворуч

Сцена доповненої реальності на **WebXR + three.js** із двома GLTF-моделями,
розміщеними за **відносними віртуальними координатами** глядача:

| Модель | Положення | Координати (X, Y, Z) |
|--------|-----------|----------------------|
| 🦩 Фламінго | **ліворуч** | (−1, 1.2, −2) |
| 🦜 Папуга | **праворуч** | (+1, 1.2, −2) |

Глядач перебуває у точці (0, 0, 0) і дивиться вздовж осі −Z, тому:
- від'ємний X — **ліворуч**
- додатний X — **праворуч**
- від'ємний Z — **перед глядачем**

## Запуск

- **На телефоні з підтримкою WebXR** (Android + Chrome): натисніть «Enter AR».
- **На комп'ютері**: сцена відкривається у звичайному 3D-перегляді —
  обертайте мишею, щоб побачити моделі ліворуч і праворуч.

## Файли

- `index.html` — розмітка та підключення three.js
- `main.js` — логіка сцени, завантаження моделей, WebXR

Моделі завантажуються з офіційного репозиторію three.js
(`raw.githubusercontent.com/mrdoob/three.js`).
